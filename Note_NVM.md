# NVM Configuration

***
##  1. Installing
### 1.1 Installing with brew
```
$ brew install nvm
```

### 1.2 zsh config
Edit `~/.zshrc`
Add:
```
export NVM_DIR=~/.nvm
source $(brew --prefix nvm)/nvm.sh
```
Take Effect: `$ source ~/.zshrc`

## 2. Usage

### 2.x Install yarn
```
$ npm install -g yarn
$ npm install -g @vue/cli
```
